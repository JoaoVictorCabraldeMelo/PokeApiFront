# PokeTrader

## Url
- https://poketrader.netlify.app/

## Como Rodar

Entre no repositorio do projeto e rode
```bash
npm install
```

e depois rode

```bash
npm start
```

## Como Rodar Testes

No repositorio do projeto rode 

```bash
npm test
```
